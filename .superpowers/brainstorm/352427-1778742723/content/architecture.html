Graph View — Architecture & Layout

A new "Graph" tab with three visualization modes, powered by GitNexus + git data

Architecture Overview

Data Flow & Component Tree
Server — New Routes
GET /api/graph/clusters
GET /api/graph/dependencies
GET /api/graph/process
GET /api/graph/tree
Wraps GitNexus Cypher queries, git ls-tree, and git log --stat into graph-ready JSON
Client — Data Layer (src/services/graphData.ts)
fetchClusterGraph()
fetchDependencyGraph()
fetchProcessFlow()
fetchTreeMap()
Transforms API responses into Cytoscape-compatible elements[] and layout configs
Client — React Components
GraphView
Top-level tab container
Mode switcher, search bar
GraphCanvas
Cytoscape component
Zoom, pan, node clicks
GraphSidebar
Node detail panel
Symbol info, nav links
Rendering — cytoscape + react-cytoscapejs
Layouts: cose-bilkent (clusters), breadthfirst (dependencies), dagre (process flows)
